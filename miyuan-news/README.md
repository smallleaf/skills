本技能是用来查询某个网址的新闻信息，然后输出新闻总结。这只是一个示例。
但是如果发挥你自己的聪明才智，你就能发现，可以用这种方式做非常多的事情。

如果你发现你的工作，跟浏览器频繁打交道，就可以用这种方式来做。

如何去利用这个脚本做一个新的skill呢。请参考以下对话：
你的需求：
统计A和B网址的，某个数据信息。

你给openclaw如下对话：
请参考miyuan-news技能，给我创建一个新的技能，名称：xxx。需求是我需要统计A和B网址的，某个数据信息。网址地址是：，信息是：。如果你有示例当然就最好。
接下来就耐心，等待openclaw帮你创建好技能。创建好技能，就可以等待执行验证就可以了。


一但，我们发现了，openclaw可以浏览你的浏览器了，那么他是不是可以替代你去做浏览器相关工作了，太可怕了！
