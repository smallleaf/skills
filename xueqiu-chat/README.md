# xueqiu-chat — 雪球股票讨论抓取助手

## 功能

输入股票代码，自动打开浏览器抓取雪球最新用户讨论内容，总结成简洁的市场情绪摘要。

## 触发场景

- "SH600026 最近在聊什么"
- "000001 雪球上的讨论"
- "帮我看看 xxx 股票的市场情绪"

## 股票代码格式

| 输入格式 | 处理方式 |
|----------|----------|
| `SH600026` / `SZ000001` | 直接使用（统一转大写） |
| `600026`（纯数字） | 6开头补 `SH`，0/3开头补 `SZ` |
| `sh600026`（小写） | 自动转大写 |

## 输出内容

- 最新讨论热点
- 用户情绪倾向（看多/看空/中性）
- 高频关键词
- 代表性帖子摘要

## 依赖

- `playwright`：`pip install playwright && playwright install chromium`
- `openclaw browser` CLI

## 数据来源

[雪球](https://xueqiu.com)
