# 雪球讨论分析引擎 (Xueqiu Discussion Analyzer)

## 🎯 目标
模拟打开浏览器访问 `https://xueqiu.com/S/{code}`，抓取最新用户讨论，总结成简洁的市场情绪摘要。

---

## 📋 执行流程

### Step 1：识别股票代码
从用户输入中识别股票代码：
- 已有前缀：`SH600026` / `SZ000001` → 直接使用（统一转大写）
- 只有数字：`600026` → 6开头补 `SH`，0/3开头补 `SZ`
- 大小写混合：`sh600026` → 自动转大写

---

### Step 2：环境检测
```bash
python3 -c "from playwright.sync_api import sync_playwright; print('OK')" 2>/dev/null || echo "MISSING"
```
- ✅ OK：继续 Step 3
- ❌ MISSING：直接进入降级策略

---

### Step 3：运行爬虫
```bash
python3 ~/.openclaw/workspace/skills/xueqiu-chat/scripts/xueqiu_crawler.py \
  --code [股票代码] \
  --output /tmp
```

爬虫行为：
1. 先访问雪球首页建立真实 session/cookie
2. 模拟人类鼠标移动和随机延迟
3. 跳转到 `https://xueqiu.com/S/{code}`
4. 关闭登录弹窗（如有）
5. 滚动页面加载更多讨论
6. **直接读取 `page.content()`（完整 HTML）+ `page.inner_text("body")`（干净文本）**
7. 解析合并为完整帖子，保存到 `/tmp/xueqiu_{code}.json`

> ⚠️ 雪球有反爬机制，爬虫使用非无头模式 + JS指纹隐藏 + 随机延迟应对。
> 若仍被拦截（name 为"访问验证"），进入降级策略。

---

### Step 4：读取结果并判断
```bash
cat /tmp/xueqiu_{code}.json
```

检查 `stock_info.name` 是否为"访问验证"：
- 正常 → 继续 Step 5
- 被拦截 → 进入降级策略

---

### Step 5：总结输出

基于抓取到的讨论内容，生成以下格式的摘要：

---

## 📄 输出格式

```
📈 [股票名称]（代码）讨论摘要

💹 当前价格：XX.XX  涨跌幅：±X.XX%

📢 最新讨论热点（共 N 条）

🔥 主要观点：
• [观点1 - 简洁提炼，不超过20字]
• [观点2]
• [观点3]

😊 市场情绪：[偏乐观 / 偏悲观 / 分歧较大 / 观望为主]

💬 典型发言：
> "[原文摘录1，不超过50字]"
> "[原文摘录2，不超过50字]"

⚠️ 以上内容仅为讨论摘要，不构成投资建议。
⏱ 数据时间：YYYY-MM-DD HH:mm
```

---

## 🔁 降级策略（被反爬拦截时）

1. 直接告知用户："雪球触发了访问验证，无法自动抓取。"
2. 引导用户手动操作：
   - 打开 `https://xueqiu.com/S/{code}`
   - 复制页面讨论内容粘贴给我
3. 收到用户粘贴内容后，按输出格式整理总结

---

## ⚠️ 核心约束

1. **只总结真实内容** — 不编造任何讨论观点
2. **情绪判断客观** — 基于讨论词语倾向判断，不掺杂个人观点
3. **投资免责** — 输出末尾必须注明不构成投资建议
4. **代码容错** — 自动处理大小写、自动补全交易所前缀
